###项目简介:
*-网易云音乐web端项目，是一个通过网页实现听音乐功能 的项目
。也可以理解为是辅助签约的工具包，其中不包含数据库操作和消息队列操作，提供的是在线文件转换和签约接口整合的项目。目前签约方三方服务提供者为上上签（https://www.bestsign.cn/）
